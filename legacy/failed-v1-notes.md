# Failed v1 Archive Notes

On 1 July 2026, the original Complete Cruising application line was frozen for the fresh-base rescue on branch `fresh-base/sun-princess-2026`.

## Archived paths

- `legacy/failed-v1-app-src`
- `legacy/failed-v1-public`
- `legacy/failed-v1-app-README.md`

## Reason

The original implementation had become too complex and too fragile for the real Sun Princess August 2026 cruise. Import workflows, local database behaviour, backstage tooling and rescue controls had started to dominate the product while still failing to provide a dependable guidebook.

## What remains active

The active deployed candidate is now the fresh-base app under `app/src`, built as a static, cruise-stable companion for one sailing only.

## What this archive is for

- visual and wording reference
- factual seed ideas
- route and page inspiration
- historical record of the failed implementation

It must not be treated as the architectural base for future work on this branch.
