# Fresh-Base Product Brief

## Objective

Create a premium, static, cruise-stable companion for the Lawrence family's Sun Princess Mediterranean 2026 sailing.

## In scope

- Dashboard
- Today
- Itinerary
- Ports
- Ship
- Plans
- Memories
- About

## Out of scope

- imports
- local database flows
- backstage tooling
- reset and data-management screens
- authentication, backend or paid APIs

## Product qualities

- British English
- Ocean Luxe presentation
- mobile-first readability
- calm confidence language
- cruise-specific usefulness within seconds
