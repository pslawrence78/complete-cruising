# 0001: Project start

**Status:** Accepted  
**Date:** 19 June 2026

## Decision

Complete Cruising will begin as a local-first static PWA suitable for GitHub Pages deployment.

- Preserve the standalone HTML prototype as the authoritative Ocean Luxe design reference, not production code.
- Deliver the project through controlled, independently reviewable tranches.
- Avoid live APIs and API keys in the MVP.
- Keep real booking details, private family information and other sensitive data out of public sample files.

## Consequences

Early development can focus on a premium offline-capable experience and trusted local data without introducing backend, authentication or live-integration complexity.
